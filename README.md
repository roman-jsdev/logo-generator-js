# Vanilla JS Logo Generator App
A useful and free application which allows creating a unique and  
beautiful logo for companies through a fascinating quiz.

![alt text](https://github.com/romanjsprodev/logo-generator-js/blob/master/src/thumb.PNG)
## Main Features
- Component-based single page application
- Unique component rendering system
- Framework-free app
- Option for downloading logos
- Preserving favourite logos
- Fully responsive design and functionality
## Demo
Discover all the features of the application absolutely free and right here [Demo](https://js-logo-generator.web.app/)
